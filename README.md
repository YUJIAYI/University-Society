# 校园活动交流平台 University-Society

* 主要功能  
    * 普通用户端（微信小程序）  
        * 用户申请认证 
        * 浏览活动列表 
        * 报名、收藏活动 
        * 二维码签到 
        * 活动反馈等 
    * 组织管理端（PC）
        * 增删改组织内部权限的活动信息
        * 实时查看活动报名情况
    * 学生认证端（PC）
        * 对认证申请进行审核工作
* 主要技术  
    * 前端主要使用weui框架
    * 后台使用.net mvc
    * 数据库为sql sever
